Primary UX Bet

Users should create Vocabulary from transcript mistakes, because Settings-first vocabulary management makes the user invent examples instead of correcting real failures.

Interaction Detail

The sheet includes Save and Re-clean because immediate repair is valuable, but that may be the main implementation scope decision for v1.

Boundary

Vocabulary only affects Cleanup. If Cleanup is off, Vocabulary is stored but inactive, which keeps routing and privacy behavior explainable.
