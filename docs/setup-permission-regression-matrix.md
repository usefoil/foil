# Setup Permission Regression Matrix

This matrix is the release oracle for Foil setup permission regressions. Every
known failure mode must have automated coverage, documented manual QA, or a
queued follow-up task before release sign-off.

Status values:

- `covered_auto`: enforced by unit or UI tests.
- `covered_manual`: enforced by a release QA runbook or checklist.
- `missing`: no proof exists and the row must be fixed before sign-off.
