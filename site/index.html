Foil

Push-to-talk speech to text for macOS.

Hold a key, speak, release. Foil transcribes instantly and pastes text wherever your cursor is, using Groq, local whisper.cpp, or any OpenAI-compatible endpoint.

How it works

⌘ Space

Hold

Press and hold your chosen hotkey.

Speak

Talk naturally while Foil listens.

Space

Release

Let go and Foil transcribes.

Paste

Text appears wherever your cursor is.

Choose your provider

Use the transcription path that fits your workflow.

Groq Whisper

Fast hosted transcription with a Groq API key.

  • Quick setup
  • Good fit for daily dictation
  • Optional cleanup modes

Local whisper.cpp

Run an OpenAI-compatible local server on your Mac.

  • No hosted transcription required
  • Works with local resources
  • Copyable setup commands

OpenAI-compatible

Point Foil at the endpoint and model you already trust.

  • Bring your own endpoint
  • Optional API keys
  • Flexible model routing
